# Личный кабинет студента
## Описание
Веб-приложение для студентов и преподавателей, обеспечивающее удобный доступ к учебным материалам, расписанию, оценкам и заданиям.

## Основной функционал
- Просмотр расписания занятий
- Просмотр оценок и успеваемости
- Загрузка и сдача заданий
- Управление личным профилем
- Просмотр учебных материалов

## Технологии
- Java/Spring Boot (бэкенд)
- HTML/CSS/JavaScript (фронтенд)
- MySQL (база данных)
- Git (контроль версий)
